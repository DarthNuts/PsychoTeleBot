# Получение API-ключа OpenRouter

OpenRouter — сервис-агрегатор ИИ-моделей. Через один API-ключ доступны сотни моделей, включая бесплатные.

---

## 1. Регистрация

1. Перейдите на сайт: https://openrouter.ai
2. Нажмите **Sign In** (правый верхний угол)
3. Войдите через Google, GitHub или email

---

## 2. Создание API-ключа

1. После входа перейдите на страницу ключей: https://openrouter.ai/keys
2. Нажмите **Create Key**
3. Введите название ключа (например, `PsychoTeleBot`)
4. Нажмите **Create**
5. Скопируйте ключ — он начинается с `sk-or-v1-...`

> **Важно:** ключ показывается только один раз. Если потеряли — создайте новый.

---

## 3. Добавление ключа в проект

Откройте файл `.env` в корне проекта и добавьте:

```env
OPENROUTER_API_KEY=sk-or-v1-ваш_скопированный_ключ
```

---

## 4. Выбор модели

В `.env` укажите модель:

```env
OPENROUTER_MODEL=arcee-ai/trinity-large-preview:free
```

### Бесплатные модели

Бесплатные модели имеют суффикс `:free`. Они не тратят кредиты, но могут иметь ограничения по количеству запросов в минуту.

Актуальный список: https://openrouter.ai/models?q=free

Рекомендуемые:

| Модель | Описание |
|--------|----------|
| `arcee-ai/trinity-large-preview:free` | Большая модель, 131K контекст, хорошо работает с русским |
| `qwen/qwen3-next-80b-a3b-instruct:free` | Мощная (80B параметров), отлично для русского, 262K контекст |
| `meta-llama/llama-3.2-3b-instruct:free` | Лёгкая и быстрая |

Чтобы найти доступные модели из консоли:
```bash
python list_openrouter_models.py
```

### Платные модели

Для платных моделей (без суффикса `:free`) нужно пополнить баланс: https://openrouter.ai/credits

Новые аккаунты могут получить $1–2 бесплатных кредитов.

Примеры:

| Модель | Цена (за 1M токенов) |
|--------|---------------------|
| `openai/gpt-4o-mini` | ~$0.15 |
| `google/gemini-flash-1.5` | ~$0.075 |
| `anthropic/claude-3.5-sonnet` | ~$3.00 |

Актуальные цены: https://openrouter.ai/models

---

## 5. Проверка

Запустите бота и отправьте сообщение в режиме «Консультация с ИИ». Если бот отвечает — ключ работает.

Если ИИ не отвечает, проверьте:

- Правильность ключа в `.env` (без пробелов и кавычек)
- Доступность выбранной модели (бесплатные модели могут быть временно недоступны)
- Баланс кредитов (для платных моделей): https://openrouter.ai/credits
- Логи бота — ошибки API выводятся в консоль
